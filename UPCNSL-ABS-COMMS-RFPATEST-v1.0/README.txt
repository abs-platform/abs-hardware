README

The software used is Altium Designer v14.3.

Fabrication requirements: FR4 substrate.

UPCNSL-ABS-RF-POL includes the designs of RF and EPS.

Fabrication Files includes the Gerber files to fabricate the boards of RF and EPS.

Components List RF_EPS includes the components of the designed boards.

The file with the name UPCNSL-ABS-COMMS-RFPATEST-v1.0 includes the designs of the RF hardware subsystem:
	-Schematic file.
	-PCB file.
	-PDF with the schematic and layout.
